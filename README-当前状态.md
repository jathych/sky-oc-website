## 📝 快速操作指南

### 启动网站
```bash
cd oc-website
npm run dev
```
访问：http://localhost:3000

### 编辑内容
1. **添加小剧场/正剧内容**
   - 编辑：`oc-website/data/stories.ts`
   - 参考：`如何添加内容.md`

2. **修改角色信息**
   - 编辑：`oc-website/data/characters.ts`

3. **添加图片**
   - 角色图：`oc-website/public/images/characters/`
   - 作品图：`oc-website/public/images/artworks/`

### 测试网站
- 保存文件后，浏览器会自动刷新
- 如果出现错误，检查控制台输出
