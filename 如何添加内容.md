# 如何从 Word 文档添加内容到网站

## 方法一：手动复制粘贴（推荐）

### 添加小剧场内容

1. 打开 Word 文档（如 `番外 人鱼传说.docx`）
2. 选择并复制全部内容（Ctrl+A，然后 Ctrl+C）
3. 打开 `oc-website/data/stories.ts` 文件
4. 找到对应的小剧场条目，例如：
   ```typescript
   {
     id: 'scene-mermaid',
     title: '番外 人鱼传说',
     type: 'scene',
     order: 1,
     description: '一个关于人鱼的奇幻故事',
     content: '内容待添加 - 请从 番外 人鱼传说.docx 中复制粘贴内容到这里',
   },
   ```
5. 将 `content` 字段的内容替换为你复制的文本
6. 注意保持文本格式，使用反引号包裹：
   ```typescript
   content: `这里是从Word复制的内容
   可以包含多行文本
   段落之间会自动保留
   `,
   ```

### 添加正剧章节

1. 打开 `正剧.docx` 文档
2. 确定章节划分（按标题或内容划分）
3. 逐个复制每个章节的内容
4. 打开 `oc-website/data/stories.ts` 文件
5. 在 `mainNovel.chapters` 数组中添加章节：
   ```typescript
   export const mainNovel: MainNovel = {
     title: '光与影的旅程',
     description: '箬笠与公主在光之子与暗之子对立的世界中，寻找真相与自我的冒险故事。',
     chapters: [
       {
         id: 'chapter-1',
         title: '第一章 墓土初遇',
         content: `这里粘贴第一章的内容...`,
         publishDate: '2025-01-09',
       },
       {
         id: 'chapter-2',
         title: '第二章 同行之路',
         content: `这里粘贴第二章的内容...`,
         publishDate: '2025-01-10',
       },
       // 继续添加更多章节...
     ],
   };
   ```

## 方法二：使用在线转换工具

如果你的 Word 文档比较复杂（包含格式、图片等），可以：

1. 使用在线工具将 .docx 转换为纯文本：
   - https://convertio.co/zh/docx-txt/
   - https://www.aconvert.com/cn/document/docx-to-txt/

2. 下载转换后的 .txt 文件
3. 打开 .txt 文件，复制内容
4. 按照方法一粘贴到对应位置

## 注意事项

### 1. 特殊字符处理
- 如果内容中包含反引号 \`，需要转义：\\`
- 如果内容中包含 ${ }，需要转义：\\${ }
- 双引号 " 需要转义：\\"

### 2. 保持段落格式
使用模板字符串（反引号 \` \`）可以保持原有的换行和段落：
```typescript
content: `第一段内容

第二段内容

第三段内容`,
```

### 3. 长文本建议
如果单个文件内容很长（超过 1000 行），建议：
- 分章节添加
- 每次保存后测试网站是否正常显示

## 示例：完整的小剧场添加流程

### 步骤 1：打开 Word 文档
打开 `番外 人鱼传说.docx`

### 步骤 2：复制内容
全选内容（Ctrl+A），复制（Ctrl+C）

### 步骤 3：编辑数据文件
打开 `oc-website/data/stories.ts`，找到：

```typescript
{
  id: 'scene-mermaid',
  title: '番外 人鱼传说',
  type: 'scene',
  order: 1,
  description: '一个关于人鱼的奇幻故事',
  content: '内容待添加',
},
```

### 步骤 4：粘贴并格式化
改为：

```typescript
{
  id: 'scene-mermaid',
  title: '番外 人鱼传说',
  type: 'scene',
  order: 1,
  description: '一个关于人鱼的奇幻故事',
  content: `这里粘贴从Word复制的内容

保持原有的段落格式
可以包含多行

故事继续...`,
},
```

### 步骤 5：保存并测试
1. 保存文件（Ctrl+S）
2. 如果开发服务器正在运行，网站会自动刷新
3. 访问 http://localhost:3000/scenes 查看小剧场列表
4. 点击对应的小剧场查看完整内容

## 快捷编辑建议

使用支持语法高亮的编辑器（如 VS Code）来编辑 `.ts` 文件，这样可以：
- 自动检测语法错误
- 更容易看到字符串的开始和结束
- 使用自动格式化功能

## 遇到问题？

### 问题 1：保存后网站显示错误
- 检查是否有未闭合的引号或反引号
- 确保 TypeScript 语法正确（逗号、括号等）

### 问题 2：内容显示不完整
- 检查是否有特殊字符需要转义
- 确认 content 字段使用了反引号 \` \`

### 问题 3：段落格式丢失
- 使用反引号模板字符串而不是普通引号
- 保留 Word 文档中的空行
