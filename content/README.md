# Content 文件夹使用说明

## 这个文件夹是什么？

这个文件夹用来存放网站的所有文本内容。你只需要将 Word 文档的内容复制粘贴到对应的 .txt 文件中，网站就会自动读取显示。

## 文件夹结构

```
content/
├── scenes/              # 小剧场内容
│   ├── 人鱼传说.txt
│   ├── 神女与她虔诚的信徒.txt
│   └── 小红狼与大灰帽.txt
└── chapters/            # 正剧章节（按需添加）
    ├── 第1章.txt
    ├── 第2章.txt
    └── ...
```

## 如何使用

### 方法一：直接复制粘贴（推荐）

1. 打开 Word 文档（如 `番外 人鱼传说.docx`）
2. 全选内容（Ctrl+A）
3. 复制（Ctrl+C）
4. 打开对应的 .txt 文件（如 `content/scenes/人鱼传说.txt`）
5. 全选文件内容（Ctrl+A），粘贴（Ctrl+V）
6. 保存文件（Ctrl+S）
7. 网站会自动更新内容！

### 方法二：Word 另存为 txt

1. 打开 Word 文档
2. 点击 文件 → 另存为
3. 选择文件类型：纯文本 (*.txt)
4. 保存到对应的 content 文件夹
5. 确保文件名正确

## 添加正剧章节

如果要添加正剧章节：

1. 在 `content/chapters/` 文件夹中创建新的 .txt 文件
2. 文件名格式：`第1章.txt`、`第2章.txt` 等
3. 将对应章节的内容粘贴进去
4. 网站会自动识别并显示

## 注意事项

- txt 文件使用 UTF-8 编码
- 保持段落之间的空行会让阅读更舒适
- 不要担心格式问题，网站会自动处理换行和段落
- 修改文件后，浏览器会自动刷新显示新内容

## 快速开始

现在就试试吧：

1. 打开 `番外 人鱼传说.docx`
2. 复制全部内容
3. 粘贴到 `content/scenes/人鱼传说.txt`
4. 保存
5. 在浏览器访问 http://localhost:3000/scenes 查看效果！
