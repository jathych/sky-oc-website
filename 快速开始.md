# 🚀 快速开始指南

## 启动网站

```bash
cd oc-website
npm run dev
```

然后访问：http://localhost:3000

---

## 📝 添加内容（超简单！）

### 方法1：添加小剧场

1. 将你的文件放到 `content/scenes/` 目录
   - 支持格式：`.txt` 或 `.docx`

2. 编辑 `data/stories.metadata.ts`，添加：

```typescript
{
  id: 'scene-new',
  title: '番外 新故事',
  type: 'scene',
  order: 4,
  description: '故事简介',
  contentFile: '新故事.txt',  // 你的文件名
},
```

3. 保存并刷新网站！✅

### 方法2：更新正剧

1. 直接编辑 `content/正剧.txt` 文件

2. 使用章节格式：
```
第一章：标题
正文内容...

第二章：标题
正文内容...
```

3. 保存后刷新网站，章节自动显示！✅

---

## 🎨 调整网站样式

1. 访问网站底部的"主题管理"链接
   - 或直接访问：http://localhost:3000/admin/theme

2. 选择你喜欢的颜色

3. 点击"保存配置"

4. 刷新页面查看效果！✅

---

## 📂 重要文件位置

```
内容文件：
📁 content/
  ├── scenes/          ← 小剧场文件放这里
  └── 正剧.txt         ← 正剧内容

配置文件：
📁 data/
  └── stories.metadata.ts  ← 配置内容元数据

主题配置：
📁 config/
  └── theme.config.json   ← 颜色主题配置
```

## ⚡ 常用命令

```bash
# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm run start
```

---

## 💡 提示

- **不需要重启服务器**：修改内容文件后，刷新浏览器即可
- **支持Word文档**：可以直接使用 `.docx` 文件
- **章节自动识别**：正剧会自动提取章节标题
