# 光遇世界 OC 网站使用说明

这是一个基于 Next.js 构建的原创角色展示网站，用于展示光遇世界观中的角色、小剧场和正剧故事。

## 🚀 快速开始

### 启动开发服务器

```bash
cd oc-website
npm run dev
```

访问 http://localhost:3000 查看网站

### 构建生产版本

```bash
npm run build
npm start
```

## 📁 项目结构

```
oc-website/
├── app/                      # Next.js 应用目录
│   ├── page.tsx             # 首页
│   ├── characters/          # 角色页面
│   ├── scenes/              # 小剧场页面（3个番外）
│   ├── novel/               # 正剧页面
│   └── search/              # 搜索页面
├── components/              # React 组件
│   ├── Navigation.tsx       # 导航栏
│   └── Footer.tsx           # 页脚
├── data/                    # 数据文件
│   ├── characters.ts        # 角色数据
│   └── stories.ts           # 故事数据（小剧场+正剧）
├── public/                  # 静态资源
│   └── images/
│       ├── characters/      # 角色图片
│       └── artworks/        # 作品图片
└── 如何添加内容.md          # 详细的内容添加指南
```

## 📝 如何添加内容

### 1. 添加新角色

编辑 `data/characters.ts` 文件：

```typescript
{
  id: 'character-id',           // 唯一标识符
  name: '角色名',
  nickname: '昵称',             // 可选
  faction: 'light',             // 'light' | 'dark' | 'neutral'
  image: '/images/characters/角色名.jpg',
  description: '角色简介',
  details: {
    age: '年龄',
    occupation: '职业',
    personality: '性格',
    background: '背景故事',
  }
}
```

然后将角色图片放到 `public/images/characters/` 目录下。

### 2. 添加小剧场

目前网站有3个小剧场：
- 番外 人鱼传说
- 番外 神女与她虔诚的信徒
- 番外 小红狼与大灰帽

要添加内容，请查看 `如何添加内容.md` 文件，里面有详细的步骤说明。

### 3. 添加正剧章节

编辑 `data/stories.ts` 文件，在 `mainNovel.chapters` 数组中添加：

```typescript
{
  id: 'chapter-x',
  title: '第X章 章节标题',
  content: '章节正文...',
  publishDate: '2025-01-09',    // 可选
}
```

### 5. 添加图片

- **角色图片**：放到 `public/images/characters/` 目录
- **作品图片**：放到 `public/images/artworks/` 目录

图片格式建议：JPG 或 PNG，文件名使用中文或英文均可。

## 🎨 主题定制

主题颜色配置在 `tailwind.config.ts` 文件中：

```typescript
colors: {
  light: {
    primary: "#FFD89B",    // 光之子主色 - 金色
    secondary: "#87CEEB",  // 光之子副色 - 天蓝
    // ...
  },
  dark: {
    primary: "#8B4C6B",    // 暗之子主色 - 深紫红
    secondary: "#2D2D44",  // 暗之子副色 - 深灰紫
    // ...
  }
}
```

修改这些颜色值即可改变整个网站的主题色调。

## 🔍 搜索功能

网站内置了搜索功能，可以搜索：
- 角色名称、昵称、简介
- 故事标题、简介、内容
- 正剧章节

搜索入口在导航栏的搜索按钮。

## 📱 响应式设计

网站已经适配了桌面端、平板和手机端，会自动根据屏幕大小调整布局。

## 💡 提示

1. **文档转换**：你的 .docx 文档中的内容需要复制粘贴到数据文件中
2. **图片优化**：建议压缩图片以提高加载速度
3. **备份数据**：定期备份 `data/` 目录下的文件
4. **开发建议**：
   - 使用 `npm run dev` 进行开发
   - 每次修改数据文件后刷新浏览器即可看到效果
   - 如果添加了新的图片，需要重启开发服务器

## 🆘 常见问题

**Q: 如何修改网站标题？**
A: 编辑 `app/layout.tsx` 中的 `metadata` 对象。

**Q: 图片不显示怎么办？**
A: 检查图片路径是否正确，图片文件是否存在于 `public/images/` 目录下。

**Q: 如何关闭开发服务器？**
A: 在命令行按 `Ctrl + C`。

**Q: 如何修改主题颜色？**
A: 编辑 `tailwind.config.ts` 文件中的颜色配置。
