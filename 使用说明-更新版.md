## 📁 文件管理

### 添加小剧场内容

**步骤：**

1. 将你的文件（`.txt` 或 `.docx`）放到 `oc-website/content/scenes/` 目录下

2. 编辑 `oc-website/data/stories.ts` 文件，找到小剧场元数据部分：

```typescript
const scenesMetadata = [
  {
    id: 'scene-mermaid',
    title: '番外 人鱼传说',
    type: 'scene' as const,
    order: 1,
    description: '一个关于人鱼的奇幻故事',
    contentFile: '人鱼传说.txt',  // 改成你的文件名
  },
  // 添加更多小剧场...
];
```

3. 保存文件，刷新网站即可看到效果！

**支持的文件格式：**
- `.txt` - 纯文本文件
- `.docx` - Word文档（系统会自动提取文本）

### 添加正剧内容

**步骤：**

1. 将正剧文件（`.txt` 或 `.docx`）放到 `oc-website/content/` 目录下

2. 编辑 `oc-website/data/stories.ts` 文件，找到正剧配置：

```typescript
const mainNovelConfig = {
  title: '光与影的旅程',
  description: '箬笠与公主在光之子与暗之子对立的世界中，寻找真相与自我的冒险故事。',
  sourceFile: '正剧.txt',  // 你的正剧文件名
};
```

3. **章节自动识别：** 系统会自动识别章节标题！
   - 支持格式：`第一章：标题`、`第1章 标题`、`第二章 标题` 等
   - 如果文件中没有章节标记，整个文件会作为一章显示

4. 保存后刷新网站，正剧章节会自动显示！

## 🎨 主题管理

### 访问主题管理页面

1. 在网站底部点击"主题管理"链接
2. 或直接访问：`http://localhost:3000/admin/theme`

### 可以调整的内容

**颜色配置：**
- **光之子配色**：主色调、次色调、强调色、文字颜色
- **暗之子配色**：主色调、次色调、强调色、文字颜色
- **中性配色**：背景、卡片、边框颜色

**渐变效果：**
- 光之子渐变
- 暗之子渐变

**网站信息：**
- 网站名称
- 网站描述

### 使用方法

1. 点击颜色框可以使用颜色选择器
2. 或直接输入十六进制颜色代码（如 `#FFD89B`）
3. 修改完成后点击"保存配置"
4. **刷新页面查看效果**

### 恢复默认

如果不满意当前配置，点击"重置为默认"按钮即可恢复原始主题。

## 📝 快速开始指南

### 1. 启动网站

```bash
cd oc-website
npm run dev
```

访问：`http://localhost:3000`

### 2. 添加新内容的完整流程

**示例：添加一个新的小剧场**

1. **准备文件**
   - 在Word或文本编辑器中写好内容
   - 保存为 `新故事.txt` 或 `新故事.docx`

2. **放置文件**
   - 将文件复制到 `oc-website/content/scenes/` 目录

3. **更新配置**
   - 打开 `oc-website/data/stories.ts`
   - 在 `scenesMetadata` 数组中添加：
   ```typescript
   {
     id: 'scene-new-story',
     title: '番外 新故事',
     type: 'scene' as const,
     order: 4,
     description: '这是一个新的故事',
     contentFile: '新故事.txt',
   },
   ```

4. **查看效果**
   - 保存文件
   - 网站会自动刷新
   - 访问 `/scenes` 查看新添加的小剧场

### 3. 自定义网站外观

1. 访问 `http://localhost:3000/admin/theme`
2. 调整你喜欢的颜色
3. 点击"保存配置"
4. 刷新页面查看效果

## ⚠️ 注意事项

### 文件编码
- 请使用 UTF-8 编码保存文本文件
- Word文档会自动处理编码问题

### 文件名
- 文件名中避免使用特殊字符
- 建议使用中文或英文字母、数字

### 章节过长
- 如果单章内容超过10000字，建议分成多章
- 可以提高阅读体验和加载速度

### 主题配置
- 修改主题后需要刷新页面
- 颜色对比度要合理，确保文字可读性
- 建议先在小范围测试，满意后再全局应用

## 📞 常见问题

### Q: 添加新文件后看不到内容？
A: 检查以下几点：
- 文件是否放在正确的目录（`content/scenes/` 或 `content/`）
- `stories.ts` 中的 `contentFile` 是否正确
- 文件编码是否为UTF-8
- 刷新浏览器缓存（Ctrl+F5）

### Q: 主题保存后没有生效？
A:
- 确认点击了"保存配置"按钮
- 刷新页面（F5 或 Cmd+R）
- 查看浏览器控制台是否有错误

### Q: 正剧章节识别不正确？
A:
- 检查章节标题格式是否符合要求
- 确保章节标题独占一行
- 如果还有问题，可以手动在 `stories.ts` 中定义章节

### Q: 如何添加图片？
A:
- 将图片放到 `public/images/` 目录
- 在内容文件中无法直接引用图片
- 需要在代码层面修改（未来可能会支持）
